# 📄 Türkiye’de Online İş İlanları Üzerinden Bölgesel Beceri Talebinin Analizi  
### A Text Mining and Machine Learning Approach

## 📌 About the Study
This repository contains the conference paper titled  
**“Türkiye’de Online İş İlanları Üzerinden Bölgesel Beceri Talebinin Analizi: Bir Metin Madenciliği ve Makine Öğrenmesi Yaklaşımı”**.

The study analyzes regional skill demands in Türkiye using online job postings.  
By applying text mining and machine learning techniques, it identifies patterns in demanded skills across different regions.

---

## 🧠 Abstract
This study aims to reveal regional differences in skill demands by analyzing online job postings.  
Text mining techniques were applied to extract meaningful features from job descriptions, and machine learning algorithms were used to cluster and classify the data.  
The findings provide insights into labor market dynamics and regional workforce needs.

---

## ⚙️ Methods
The following methods were used in the analysis:

- **TF-IDF (Term Frequency – Inverse Document Frequency)**  
  → Feature extraction from text data  

- **K-Means Clustering**  
  → Identifying groups of similar job postings  

- **Random Forest**  
  → Classification and prediction of skill demand patterns  

---

## 📊 Dataset
- Source: Online job postings  
- Content: Job descriptions, required skills, regional information  
- Note: Dataset may be anonymized due to privacy considerations
